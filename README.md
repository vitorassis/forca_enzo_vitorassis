# forca_enzo_vitorassis

Para rodar, clone ou faça download ZIP do repositório.

Windows (CMD):
```
cd Path\to\Folder
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

Linux/ Mac
```
cd Path/to/Directory
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
